# Header Component

## 1. Metadata
- Name: Header
- Category: Component
- Status: Active

## 2. Overview
Primary top navigation and site identity. Use on all top-level pages.
Do not use as an inline section header.

## 3. Anatomy
- Site title/brand area
- Primary nav links
- Search/utility controls
- Mobile navigation container

## 4. Tokens used
- `--color-bg-body`, `--color-text-primary`, `--color-text-secondary`
- `--space-*` for layout padding/margins
- `--motion-duration-fast`, `--motion-easing-standard`

## 5. Props/API
- Astro component, no required public props.

## 6. States
- Default, hover, active link, focus-visible, mobile-expanded

## 7. Code example
```astro
<Header />
```
