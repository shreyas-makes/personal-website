# ThemeToggle Component

## 1. Metadata
- Name: ThemeToggle
- Category: Component
- Status: Active

## 2. Overview
Switches between light and dark themes by toggling root theme class.
Do not use as a generic binary toggle.

## 3. Anatomy
- Toggle button
- Theme icon state

## 4. Tokens used
- `--color-bg-overlay`, `--color-text-secondary`, `--color-text-on-overlay`
- `--radius-pill`, `--space-*`
- `--motion-duration-fast`, `--motion-easing-standard`

## 5. Props/API
- No required props

## 6. States
- Light, dark, hover, focus-visible

## 7. Code example
```astro
<ThemeToggle />
```
